# Utility Scripts

Standalone utility scripts from [MetroWest Smart Home](https://metrowestsmarthome.com) blog tutorials.

## Scripts

| Script | Description | Tutorial |
|--------|-------------|----------|
| [cozyla-processor](photo-processing/cozyla-processor/) | Process photos for Cozyla digital frames | [Blog Post](https://metrowestsmarthome.com/2025/12/21/fixing-black-images-on-your-cozyla-digital-frame/) |

## Usage

Each script folder contains:
- `README.md` - Setup instructions and requirements
- Script file(s) with `YOUR_*` placeholders for customization

See individual README files for detailed instructions.
